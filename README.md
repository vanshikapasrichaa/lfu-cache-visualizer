LFU Cache Visualizer->

It is a web-based visualizer for the Least Frequently Used (LFU) Cache algorithm built using HTML, CSS, and JavaScript.

This project helps users understand how an LFU Cache works.

Features->

- Set cache capacity dynamically
- Perform PUT and GET operations
- Visual representation of cache contents
- Frequency bucket visualization
- Automatic LFU eviction
- Operation history and logs
- Statistics panel showing cache state
- Demo and Reset functionality

Why I Built This->

LFU Cache is a commonly asked data structure in coding interviews, but its internal working can be difficult to visualize. I built this project to strengthen my understanding of cache operations and to create an interactive tool that makes the algorithm easier to learn.

Technologies Used->

- HTML
- CSS
- JavaScript
